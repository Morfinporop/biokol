# 🎉 BioGram — Мессенджер нового поколения

Полная трансформация BioLink в современный мессенджер в стиле Telegram с твоим уникальным визуалом!

## ✨ Основные возможности

### 💬 Сообщения
- ✅ Отправка текстовых сообщений
- ✅ Чтение сообщений в реальном времени
- ✅ Статус прочтения (галочки)
- ✅ Временные метки
- ✅ Группировка по датам

### 👥 Чаты
- ✅ Список всех чатов
- ✅ Счетчик непрочитанных сообщений
- ✅ Последнее сообщение в превью
- ✅ Сортировка по времени
- ✅ Поиск пользователей

### 🎨 Дизайн
- ✅ Темная тема (как в Telegram)
- ✅ Градиенты и glass morphism
- ✅ Адаптивный интерфейс
- ✅ Плавные анимации
- ✅ Современные иконки

### 🔐 Безопасность
- ✅ Авторизация
- ✅ Приватные сообщения
- ✅ Блокировка пользователей (через админку)

## 🚀 Как пользоваться

### 1. Регистрация
```
1. Открой https://biolink.up.railway.app
2. Нажми "Войти" → "Регистрация"
3. Введи email, username, пароль
4. Пройди капчу
5. Готово!
```

### 2. Начало общения
```
1. После входа откроется мессенджер
2. В поиске введи имя пользователя
3. Выбери пользователя из списка
4. Напиши первое сообщение!
```

### 3. Поиск контактов
```
- Введи username или displayName в поиск
- Минимум 2 символа для поиска
- Кликни на пользователя для начала чата
```

## 🔧 Технические детали

### API Endpoints
```
GET  /api/chats/:userId          ← Список чатов
GET  /api/messages/:u1/:u2       ← Сообщения между пользователями
POST /api/messages               ← Отправить сообщение
GET  /api/users/search/:id       ← Поиск пользователей
```

### Обновление в реальном времени
- Чаты: каждые 5 секунд
- Сообщения: каждые 3 секунды
- (В будущем можно добавить WebSocket)

### Хранение данных
- Пользователи: `server/database.json`
- Сообщения: `server/database.json` (messages array)
- Всё сохраняется на сервере!
